# Razzkr — 3D GitHub Portfolio
GitHub-style profile with 3D particles, avatar, repos, contribution graph.

```bash
npm install && npm run dev   # http://localhost:5173
```

## Deploy
```bash
git init && git add . && git commit -m "Razzkr GitHub portfolio"
git branch -M main
git remote add origin https://github.com/Razzkr/razzkr-github.git
git push -u origin main
```
Import on vercel.com → Deploy.
